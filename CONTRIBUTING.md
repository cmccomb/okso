# Contributing to okso

Thank you for your interest in contributing to `okso`!

We have detailed documentation for contributors located in the `docs/contributor/` directory:

- **[Development](docs/contributor/development.md)**: Setting up your environment, formatting, linting, and running tests.
- **[Testing](docs/contributor/testing.md)**: Detailed guide on the Bats test suite and coverage.
- **[Platform quirks](docs/contributor/platform-quirks.md)**: Notes on macOS Bash compatibility and other platform-specific details.

Please review these guides before submitting a pull request.
